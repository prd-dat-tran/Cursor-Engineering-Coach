The Improve section identifies bad habits and helps you adopt better practices for Cursor IDE.

- [Anti-Patterns](/improve/anti-patterns/) -- Detectors across five categories with scores and actionable recommendations, backed by an editable rule engine
- [Rule Editor](/improve/rule-editor/) -- Create, edit, and live-test detection rules as markdown, with form-based or raw-source editing and AI-assisted drafting
- [Rule Playground](/improve/rule-playground/) -- Interactive REPL for the rule DSL with a field browser, function catalog, and metric list
- [Data Explorer](/improve/data-explorer/) -- Browse every request/session field, view distributions, and run ad-hoc filters
- [Skill Finder](/improve/skill-finder/) -- Find repeated prompts and matching community Cursor rules
- [Context Health](/improve/context-health/) -- Evaluate context quality and window management
