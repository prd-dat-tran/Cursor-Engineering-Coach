Cursor Engineering Coach is a Cursor extension that analyzes your AI-assisted coding sessions. It reads local Cursor workspace storage and rules files and presents detailed analytics in an interactive webview panel.

## Requirements

- **Cursor IDE** (any recent version)
- Existing Cursor session history in your local workspace storage

No API keys, accounts, or external services required. Everything runs locally.

## Quick Start

1. [Install Cursor Engineering Coach]({{< ref "getting-started/installation" >}}) by building the `.vsix` from source
2. Open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
3. Run **Cursor Engineering Coach: Open Dashboard**
4. The extension scans your local Cursor session logs and displays your analytics

The dashboard opens as a webview panel inside Cursor. Use the sidebar to navigate between views.
