The Level Up section turns your Cursor IDE usage data into a progression system with learning, achievements, and social features.

- [Learning Center](/level-up/learning/) -- Personalized quizzes and challenges built from your actual usage
- [Achievements](/level-up/achievements/) -- XP-based progression with tiered badges
- [Agentic SDLC](/level-up/sdlc/) -- Track Cursor usage across the software development lifecycle
- [Share](/level-up/share/) -- Generate shareable stat cards for your team
