# Learning Center

The Learning Center is a personalized upskilling program built from your actual Cursor usage. The skill tree adapts to your languages, dependencies, and projects, so everything you learn is directly applicable to your work.

![Learning Center](/screenshots/screen-learning.png)

## How It Works

Cursor Engineering Coach analyzes the languages and technologies you work with, then generates targeted exercises. You can explore freely, but you level up only by passing quizzes.

## Quiz Types

### Language Challenges

Multiple-choice questions about language-specific behavior. For example, a Python challenge about mutable vs. immutable types asks you to predict the output of a code snippet. Questions are tagged by difficulty (EASY, MEDIUM, HARD) and pulled from the languages and projects you actually use.

### Slop or Not

A side-by-side code comparison game. Two code snippets are shown, and you identify which one is not AI-generated "slop." Each round is tagged with a readability category and difficulty level.

## Tracking

Progress stats are shown at the top of the page:

- **Correct** and **Incorrect** counts
- **Accuracy** percentage
- **Current streak** and **best streak**
- **Difficulty level** (adapts based on your performance)
- **Learning path** progress

## Filtering

You can filter challenges by:

- **Language** -- TypeScript, Python, CSS, JSON, YAML, and more
- **Project** -- Scoped to your actual workspaces

## Focus Tips

A sidebar panel shows personalized focus tips based on your coding patterns. For example:

- "Your deepest flow tends to happen around 13:00 (score 48). Block this time for uninterrupted work."
- "14 sessions were fragmented (long pauses between prompts). Close distractions during coding sessions."

## Rewards

Completing streaks and quizzes unlocks mini-games and other rewards (like a built-in Snake game).
