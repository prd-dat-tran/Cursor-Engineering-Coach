# Achievements

The Achievements page tracks your journey to mastering Cursor IDE through an XP-based progression system.

![Achievements](/screenshots/screen-achievements.png)

## Cursor Engineer Roadmap

Your total XP is displayed alongside a progress bar showing how many achievements you have unlocked. The roadmap tracks your cumulative progress across all achievement categories.

### Summary Stats

Four headline metrics appear at the top:

- **Linux Kernels** -- A fun normalization: how many Linux kernels worth of code Cursor has generated for you
- **AI Lines of Code** -- Total estimated LoC across all sessions
- **Longest Streak** -- Your longest consecutive run of active coding days
- **Peak Reqs/Hr** -- Your highest request-per-hour burst rate

## Achievement Tiers

Achievements come in four tiers:

| Tier | Color | Difficulty |
|---|---|---|
| **Bronze** | Brown | Entry-level milestones |
| **Silver** | Gray | Sustained usage targets |
| **Gold** | Yellow | Advanced milestones |
| **Diamond** | Blue | Extreme achievements |

Each tier awards different XP amounts. Progress bars show how close you are to unlocking each achievement.

## Example Achievements

### Volume

- **10K Club** (Bronze, 15 XP) -- Sent 10,000 requests in Cursor
- **The Machine** (Gold, 50 XP) -- Sent 100,000 requests in Cursor
- **Kernel Hacker** (Gold, 50 XP) -- Generated one Linux kernel worth of Cursor-assisted code (27.8M LoC)
- **Linus v2** (Diamond, 100 XP) -- Generated 5x Linux kernels worth of Cursor-assisted code

### Consistency

- **Session Marathoner** (Silver, 25 XP) -- Started 1,000 Cursor sessions

### Breadth

- **Polyglot** (Silver, 25 XP) -- Used Cursor with 10+ programming languages

## Latest Unlock

The most recently earned achievement is highlighted with a "LATEST UNLOCK" badge and the date it was unlocked.
