# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
Code of Conduct, version 2.1.

In short: be respectful, be inclusive, and assume good intent. Harassment, discrimination, and
personal attacks are not acceptable in any project space (issues, pull requests, discussions, or
chat).

If you experience or witness behavior that violates this Code of Conduct, please open a private
issue or contact the repository maintainers directly via GitHub.

The full Contributor Covenant text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
